1. Зайди на https://github.com и создай новый репозиторий
2. Загрузите туда main.py и requirements.txt из этой папки
3. Перейди на https://render.com, нажми New → Web Service
4. Подключи GitHub и выбери репозиторий с ботом
5. Настрой:
   - Build Command: pip install -r requirements.txt
   - Start Command: python main.py
6. Нажми Deploy

Готово — бот будет работать 24/7
